# 本地资源网络共享

## 主要功能

简单的代码将本机电脑的大型文件通过HTTP网页链接进行分享。  
主要用于局域网大文件传输，公网远程下载。  

## 使用方法

 1. windows系统直接下载并在共享目录运行share.exe
 2. 根据提示设置端口和路径
 3. 使用浏览器访问或下载共享资源

> win下可以通过 **win** + **r** ,输入**ipconfig** 命令查看本机ip地址

## 示例

```bash
请输入需要监听的端口:
8080
请输入监听URL的路径:
/file_share/
```

访问 ```http://[ip]:8080/file_share/``` 即可
